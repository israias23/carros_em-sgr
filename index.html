Como usar o programa.

  1. Faça upload de uma planilha Excel (.xlsx ou .xls) com as colunas: Placa, Modelo, Status, Dias.
  2. Defina o número de dias no campo "Filtrar dias maiores que" para filtrar os resultados.
  3. Selecione um status no filtro de status para refinar os resultados (ou "Todos" para ver tudo).
  4. Aguarde o carregamento dos dados (pode levar alguns segundos).
  5. Visualize os resultados na tabela exibida.
  6. Use o botão de imprimir () para imprimir a tabela.
  7. Use o botão de baixar () para salvar a tabela como imagem.
